# hexo初始化 #hexo 
*安装配置Node.js*  
[[Windows开发工具与环境#Node.js环境 node]]

*安装配置Git*  
[[Git配置与使用#Git配置与连接 git]]

- 初始化(在二级目标目录中进行)
```
hexo init
```

- 生成静态页面
```
hexo g
```

- 启动本地预览服务
```
hexo s
```

- 清楚静态页面
```
hexo clean
```
